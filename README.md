# Hoy Trabajas

## Up service
```bash
docker compose build
docker compose up
```

## Stop service
Stop the container with Ctrl + C or, in another tab with
```bash
docker compose stop
```

## Down service
Delete the container with
```bash
docker compose stop
```

## Develompent
```bash
bundle install
rails server
```

## Run tests
```bash
rspec spec
```
